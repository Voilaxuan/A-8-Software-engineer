# Usage

使用比较简单：

1、安装flask和其他依赖

```
pip install flask

```

2、导入数据库文件TDScann.sql文件

```
mysql -u root -p

use TDScan

source TDScan.sql

```

3、运行脚本

```
python2 main.py
```

会提示http://0.0.0.0:5000，打开即可使用。
